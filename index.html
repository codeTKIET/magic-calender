Introduction

The C++ project on Calendar , In this system user interact with particular date entered by the user to perform an logical operation in order to find out the particular day for desired date entered result is displayed on the screen along with the calendar and date entered . The user can keep the record by using a text file where the operation history done by the user is stored / maintained . The tracking of the the data entered is also possible by using the text file , this system can be modified to store special events on particular date and can display on demands of the user. The calendar can be used to scheduling of daily life and along with that professional purposes. The different operation performed are as follows: Take data from user . Consider the choice entered. The choice entered either 0 or 1. Check validity of entered date/month/year. If Non-valid ask user to enter data. If Valid Calculate & Display Output. __

1.1 Objective:

In calendar user can find a particular day of desired date. This is a C++ project on a calendar which provides lot of facilities to there users . It will simplify the tasks and reduce the time required to do them.

1.2 Problem Statement:

To develop software to simplify Calendar . This project is to develop software for Calendar operation. __
